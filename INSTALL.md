# Installation Guide for Kalshi Market Finder

## Quick Setup

1. **Download the Extension**
   - All files are ready in this folder
   - No additional downloads needed

2. **Load in Chrome**
   - Open Chrome browser
   - Go to `chrome://extensions/`
   - Enable "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select this folder (`/Users/jasonrubenstein/Desktop/webten`)

3. **Test the Extension**
   - Open the `test.html` file in Chrome
   - Click the extension icon in your toolbar
   - Click "Analyze Page"
   - You should see market analysis results

## Troubleshooting

If the extension doesn't work:
1. Check Chrome's developer console for errors
2. Ensure you have internet connection
3. Try refreshing the extension in `chrome://extensions/`
4. Make sure all files are in the same folder
